## CLI

```bash
$ base --help

Commands:
  analyze   Analyze firmware → produce HardwareSpec
  synth     Synthesize HardwareSpec → component mapping
  pcb       Generate PCB (KiCad) from SynthesizedSpec
  fw        Generate firmware (bootloader, HAL, drivers, Zephyr)
  check     Validate new hardware against original traces
  evolve    Analyze bottlenecks and suggest upgrades
  pipeline  Run full pipeline: analyze → synth → pcb → fw → check → evolve

Options:
  -v, --verbose...     Verbose output
  -o, --output <PATH>  Output directory [default: output]
```
