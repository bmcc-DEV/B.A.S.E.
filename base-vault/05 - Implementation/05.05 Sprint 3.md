# Sprint 3 — PCB Generator ✅

## Objetivo

Gerar arquivos KiCad válidos a partir da netlist.

## Tarefas

- [x] Biblioteca de geração de S-expression KiCad
- [x] Gerador de esquemático (.kicad_sch)
- [x] Gerador de BOM (.csv)
- [x] Layout PCB via force-directed placement
- [x] Sistema de templates de blocos comuns
- [x] Validação com kicad-cli DRC

| Módulo | Arquivos | Status |

| sexpr | `base-pcb/src/sexpr.rs` | ✅ `SExpr` builder + `to_string()` (indentado e flat) + `escape()` + `render_kicad_file()` |
