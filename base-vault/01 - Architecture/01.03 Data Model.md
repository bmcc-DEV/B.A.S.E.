# Data Model

## Core Data Structures

```mermaid
classDiagram
    class HardwareSpec {
        +u32 version
        +String source
        +CpuArch cpu_architecture
        +u32 cpu_clock_mhz
        +Endian endianness
        +MemoryRegion[] memory_map
        +FunctionalBlock[] blocks
        +InterruptSpec[] interrupts
        +DmaChannel[] dma_channels
        +SystemConstraints constraints
        +f64 confidence
    }
    
    class FunctionalBlock {
        +BlockId id
        +BlockKind kind
        +u64 base_address
        +Register[] registers
        +Protocol protocol
        +TimingProfile timing
        +BlockId[] dependencies
    }
    
    class Register {
        +u32 offset
        +String inferred_name
        +u8 width
        +AccessType access
        +RegisterPurpose purpose
        +ObservedValue[] observed_values
        +Bitfield[] bitfields
    }
    
    class Protocol {
        +State[] states
        +Transition[] transitions
        +Condition entry_condition
        +Condition exit_condition
    }
    
    class TimingProfile {
        +LatencyRange activation
        +LatencyRange processing
        +LatencyRange interrupt_response
        +DmaLatency dma_setup
        +Duration polling_interval
    }
    
    class SystemConstraints {
        +f64 max_power_watts
        +BandwidthRequirement[] required_bandwidths
        +u32 pin_count
        +u8 pcb_layers
        +TempRange operating_temp
    }
    
    HardwareSpec "1" --> "*" FunctionalBlock

```

## TimingModel

```rust
struct LatencyRange {
    min: Duration,
    max: Duration,
    avg: Duration,
    p99: Duration,
    samples: usize,
}

struct DmaLatency {
    setup: LatencyRange,
    transfer_per_byte: LatencyRange,
    completion: LatencyRange,
}

struct TimingProfile {
    activation_latency: LatencyRange,
    processing_latency: LatencyRange,
    interrupt_response: LatencyRange,
    dma_setup_latency: Option<DmaLatency>,
    polling_interval: Option<Duration>,
}
```
