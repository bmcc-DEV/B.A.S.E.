# KiCad S-Expression Format

> **R5:** header `; engineering_draft — NOT FABRICABLE` em `.kicad_sch` / `.kicad_pcb`; title block e `gr_text` no silkscreen repetem o banner.

## Estrutura de Arquivos

| Arquivo | Conteúdo | Formato |
|---------|----------|---------|
| `project.kicad_pro` | Configuração do projeto | S-expression |
| `project.kicad_sch` | Esquemático | S-expression |
| `project.kicad_pcb` | Layout PCB | S-expression |
| `project.kicad_prl` | Arquivos de projeto | S-expression |
| `project.net` | Netlist (opcional) | S-expression |
| `bom.csv` | Bill of Materials | CSV |

## Formato do Esquemático

```lisp
(kicad_sch (version 20231121) (generator "base-pcb")
  
  ;; engineering_draft — NOT FABRICABLE
  (title_block
    (title "engineering_draft — NOT FABRICABLE")
    (date "2026-07-15")
    (rev "draft")
    (company "B.A.S.E.")
  )
  
  ;; Símbolos
  (symbol (lib_id "power:GND") (at 0 0) (unit 1)
    (in_bom yes) (on_board yes)
    (property "Reference" "#GND" (at 0 0) (effects (font (size 1.27 1.27))))
    (pin 1 (xy 0 0))
  )
  
  (symbol (lib_id "RP2350:RP2350A") (at 100 100) (unit 1)
    (in_bom yes) (on_board yes)
    (property "Reference" "U1" (at 100 50) (effects (font (size 1.27 1.27))))
    (property "Value" "RP2350A" (at 100 150) (effects (font (size 1.27 1.27))))
    (property "Footprint" "Package_QFN:QFN-56-1EP_7x7mm_P0.4mm_EP4.5x4.5mm")
    (pin 1 (xy 60 60) (name "GP0"))
    (pin 2 (xy 60 80) (name "GP1"))
  )
  
  ;; S3: labels de função UART vêm do PinDef.functions (uart0_tx / uart0_rx),
  ;; não de um netlist elétrico. Banner NOT FABRICABLE permanece no title_block.

  ;; Fios (orquestração lógica — draft)
  (wire (pts (xy 0 0) (xy 60 60)) (stroke (width 0.254) (type default)) (layer "F.Cu"))
  
  ;; Labels
  (label "uart0_tx" (at 70 60) (fields (effects (font (size 1.27 1.27)))))
  
  ;; Junções
  (junction (at 60 60))
  
  ;; Nets
  (net (code 0) (name "GND"))
  (net (code 1) (name "VCC_3V3"))
  (net (code 2) (name "GPIO5"))
)
```

## Geração Programática

```rust
fn generate_kicad_sch(components: &[PlacedComponent], nets: &[Net]) -> String {
    let mut sch = String::new();
    sch.push_str("(kicad_sch (version 20231121) (generator \"base-pcb\")\n");
    
    sch.push_str("  (title_block\n");
    sch.push_str("    (title \"B.A.S.E. Generated\")\n");
    sch.push_str(&format!("    (date \"{}\")\n", chrono::Local::now().format("%Y-%m-%d")));
    sch.push_str("  )\n");
    
    for comp in components {
        sch.push_str(&comp.to_sexpr());
    }
    for net in nets {
        sch.push_str(&net.to_sexpr());
    }
    
    sch.push_str(")\n");
    sch
}
```
