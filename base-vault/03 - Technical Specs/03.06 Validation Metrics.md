# Validation Metrics

## Métricas de Validação

| Métrica | Descrição | Fórmula | Alvo |
|---------|-----------|---------|------|
| **Pass Rate** | % de operações que passam | `passed / total × 100` | > 95% |
| **Latency Ratio** | Quanto mais rápido/lento | `t_novo / t_original` | 0.5 - 2.0 |
| **Value Accuracy** | % de valores MMIO corretos | `match / total × 100` | > 95% |
| **IRQ Coverage** | % de interrupções capturadas | `irq_novo / irq_original × 100` | > 95% |
| **DMA Throughput** | Throughput relativo | `bw_novo / bw_original × 100` | > 80% |
| **FPS Ratio** | FPS relativo (GPU) | `fps_novo / fps_original × 100` | > 75% |

## Estrutura do Relatório

```rust
struct ValidationReport {
    summary: ValidationSummary,
    mmio_results: Vec<ComparisonItem>,
    timing_profile: TimingComparison,
    interrupt_analysis: InterruptComparison,
    dma_analysis: DmaComparison,
    application_benchmarks: Vec<BenchmarkResult>,
    warnings: Vec<String>,
    failures: Vec<Failure>,
    recommendations: Vec<String>,
}

struct ValidationSummary {
    overall_pass_rate: f64,
    total_operations: usize,
    passed: usize,
    failed: usize,
    warning_count: usize,
    avg_latency_ratio: f64,
    duration: Duration,
}

struct ComparisonItem {
    operation_id: String,
    original_address: u64,
    new_address: u64,
    original_value: Option<u32>,
    actual_value: Option<u32>,
    original_latency: Duration,
    actual_latency: Duration,
    latency_ratio: f64,
    passed: bool,
    failures: Vec<String>,
}
```

## Modos de Validação

| Modo | Requer HW Real? | Velocidade | Cobertura |
|------|----------------|------------|-----------|
| **Static analysis** | Não | Segundos | Estrutural |
| **Trace replay (simulado)** | Não | Minutos | Comportamental |
| **Trace replay (hardware)** | Sim | Horas | Timing real |
| **Boot test** | Sim | Minutos | POST |
| **Application benchmark** | Sim | Horas | FPS, latência |
