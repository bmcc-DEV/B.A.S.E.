# HardwareSpec Schema

## Estrutura YAML

```yaml
# hardware_spec.yaml — schema v1
version: 1
source: "string — SpecterProbe + trace origin"

cpu:
  architecture: PowerPC | ARM | x86 | RISC-V | MIPS
  clock_mhz: 400
  endianness: Big | Little | Bi
  cores: 2

memory:
  regions:
    - name: RAM
      base: 0x00000000
      size: 0x08000000  # 128MB
      type: RAM | ROM | MMIO | Reserved
      width: 32 | 64

blocks:
  - id: gpu_0
    kind: Gpu | Audio | Dma | Usb | ...
    base_address: 0x10000000
    size: 0x1000
    
    registers:
      - offset: 0x00
        name: control
        width: 32
        access: ReadWrite | ReadOnly | WriteOnly
        purpose: Control | Status | AddressPointer | ...
        reset_value: 0x00000000
        
        observed_values:
          - value: 0x00000000
            count: 45
            context: idle
          - value: 0x00000001
            count: 12
            context: wake
            
        bitfields:
          - offset: 0
            width: 1
            name: enable
            values: { 0: "disabled", 1: "enabled" }
          - offset: 1
            width: 2
            name: mode
            values: { 0: "idle", 1: "copy", 2: "fill", 3: "reserved" }
    
    protocol:
      states: [Idle, Ready, Busy, Done, Error]
      transitions:
        - from: Idle
          to: Ready
          on: { register: control, value: 1 }
        - from: Ready
          to: Busy
          on: { register: trigger, value: 1 }
    
    timing:
      activation:
        min: 800ns
        max: 3.1us
        avg: 1.2us
        p99: 2.8us
      processing:
        min: 2.1ms
        max: 12ms
        avg: 4.5ms
        p99: 8.1ms
    
    dma:
      required: true
      min_bandwidth_mbps: 400
      alignment: 256
      max_channels: 2

interrupts:
  - vector: 16
    owner: gpu_0
    type: Level | Edge
    polarity: High | Low

constraints:
  max_power_watts: 15
  pcb_layers: 4
  temp_range: { min: 0, max: 70 }

confidence: 0.87
```

## Regras de Validação do Schema

1. Todo `block` tem `base_address` única
2. Nenhum `register` tem offset duplicado dentro do mesmo block
3. Toda `transition` refere-se a estados e registradores existentes
4. `confidence` global é calculada como média ponderada das confidences de cada bloco
