# MMU Translation Strategies

## Visão Geral

O firmware original acessa endereços MMIO específicos do ASIC. O B.A.S.E. precisa redirecionar esses acessos para os periféricos do novo hardware.

## Estratégia 1: MMU Page Remapping (Cortex-A)

```c
void setup_mmu_translation(void) {
    page_table[0x10000] = MMU_ENTRY(
        .output = 0x20000000,
        .size = MMU_4K,
        .attributes = MMU_DEVICE,
        .permissions = MMU_RW,
    );
    
    page_table[0x10010] = MMU_ENTRY(
        .output = PIO_BASE + 0x100,
        .size = MMU_4K,
        .attributes = MMU_DEVICE,
        .permissions = MMU_RW,
    );
}
```

## Estratégia 2: Software Trap Handler (sem MMU)

```c
void __attribute__((interrupt)) mmio_trap_handler(void) {
    uint64_t fault_addr = get_fault_address();
    uint32_t *fault_instr = get_fault_instruction();
    
    if (is_store(fault_instr)) {
        uint32_t value = get_store_value(fault_instr);
        uint32_t offset = fault_addr & 0xFFF;
        uint32_t base = fault_addr & ~0xFFF;
        mmio_translate_write(base, offset, value);
    } else if (is_load(fault_instr)) {
        uint32_t offset = fault_addr & 0xFFF;
        uint32_t base = fault_addr & ~0xFFF;
        uint32_t value = mmio_translate_read(base, offset);
        set_load_result(fault_instr, value);
    }
    advance_pc();
}
```

## Estratégia 3: PIO Emulation (RP2350)

```c
.program mmio_bus
.side_set 2  // WR_strobe, RD_strobe

.wrap_target
    pull block          side 0b00
    out x, 28          ; address
    out y, 32          ; data
    mov isr, x
    jp gpio_handler    ; if addr in GPIO range
    mov pins, y        side 0b10  ; WR_strobe
    nop                side 0b00
.wrap
```

## Matriz de Decisão

| Critério | MMU | Trap | PIO | Stub Driver |
|----------|-----|------|-----|-------------|
| Requer MMU | ✅ | ❌ | ❌ | ❌ |
| Overhead | 0 | ~100ns | ~20ns | ~10ns |
| Compatibilidade | Total | Total | Parcial | Baixa |
| Complexidade | Média | Alta | Média | Baixa |
| Funciona sem MMU | ❌ | ✅ | ✅ | ✅ |
| Timing exato | ✅ (se HW rápido) | ❌ | ✅ | ❌ |
