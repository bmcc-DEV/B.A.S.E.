# Playbook forense — 1 página (v0.7)

**Objetivo:** Evidence → contratos → Design (RP + STM32 multi-peripheral) + goldens + reconstruct honesto + HIL CLI EXPERIMENTAL; engenheiro no loop.

## Setup (10–12 min)

```bash
git clone https://github.com/bmcc-DEV/B.A.S.E..git
cd B.A.S.E.
cargo build -p base-cli

# Gates permanentes
./examples/pilot/run.sh
./examples/pilot/run_t1_b2.sh

# STM32 USART (Capstone + pins + goldens W2)
./examples/pilot_stm32/run.sh

# STM32 dual USART+SPI2 (W1; opt-in)
./examples/pilot_stm32/run_w1_spi.sh

# Reconstruct (estagnação honesta — ≠ auto-fix)
base reconstruct examples/pilot_stm32/out/analyze/hardware_spec.yaml \
  --continuous --threshold 0.99 -o /tmp/recon_demo
# → convergence_report.json: stop_reason / stagnated / auto_fix_complete=false

# HIL CLI (host-only)
base hil enumerate -o /tmp/hil_demo
base hil flash /dev/null --mock-flash -o /tmp/hil_demo
```

## Mostrar ao cliente

1. **RP** — UART `0x40034000` + SPI opt-in (`run_t1_b2.sh`).
2. **STM32 USART** — Capstone @ `0x40013000`; design ST → `STM32F103C8`; sch PA9/PA10 `NOT FABRICABLE`; **goldens** `diff` vs `expected/`.
3. **STM32 SPI2** — dual `@ 0x40003800` (SPI1 omitido: página 4K c/ USART1).
4. **Contratos** — replay + prove; backend no report.
5. **Reconstruct** — `--continuous` = cap 1000; para em estagnação; **não** auto-fix.
6. **HIL CLI** — Simulated / `--mock-flash`; nunca production.
