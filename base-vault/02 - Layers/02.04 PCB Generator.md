# PCB Generator (base-pcb)

## Formato Alvo: KiCad S-Expression

O KiCad 7+/8+ usa S-expressions textuais. Exemplo de um resistor:

```lisp
(symbol (lib_id "Device:R") (at 100 200) (unit 1)
  (in_bom yes) (on_board yes)
  (property "Reference" "R1" (at 50 200) (effects (font (size 1.27 1.27))))
  (property "Value" "10k" (at 150 200) (effects (font (size 1.27 1.27))))
  (property "Footprint" "Resistor_SMD:R_0805_2012Metric" (at 100 200) ...)
  (pin 1 (xy 75 200))
  (pin 2 (xy 125 200))
)
```

## Pipeline de Geração

```rust
struct PcbGenerator {
    spec: HardwareSpec,
    component_db: ComponentDb,
    templates: TemplateLibrary,
}

impl PcbGenerator {
    fn generate(&self, output_dir: &Path) -> Result<PcbOutput> {
        let components = self.select_components()?;
        let power = self.generate_power_tree(&components)?;
        let nets = self.generate_netlist(&components, &power)?;
        let placement = self.place_components(&components, &nets)?;
        let sch = self.generate_schematic(&components, &nets, &placement)?;
        let pcb = self.generate_pcb(&components, &nets, &placement)?;
        let bom = self.generate_bom(&components)?;
        self.validate(output_dir)?;
        
        Ok(PcbOutput {
            project: output_dir.join("project.kicad_pro"),
            schematic: output_dir.join("project.kicad_sch"),
            pcb: output_dir.join("project.kicad_pcb"),
            bom: output_dir.join("bom.csv"),
        })
    }
}
```

## Force-Directed Placement

```rust
struct ForceDirectedPlacer {
    iterations: usize,
    attraction: f64,
    repulsion: f64,
    damping: f64,
}

impl ForceDirectedPlacer {
    fn place(&self, components: &[Component], nets: &[Net]) -> Vec<Position> {
        // Fruchterman-Reingold adaptado para PCB:
        // 1. Nós = componentes
        // 2. Arestas = nets (peso = largura de banda)
        // 3. Atração: F_att = w * log(d / k)
        // 4. Repulsão: F_rep = k² / d
        // 5. Converge em ~100 iterações
        
        let mut positions = self.initial_positions(components);
        
        for _ in 0..self.iterations {
            let forces = self.compute_forces(components, nets, &positions);
            let temperature = self.temperature();
            for (pos, force) in positions.iter_mut().zip(forces.iter()) {
                *pos += force.normalize() * temperature.min(force.length());
            }
        }
        positions
    }
}
```

## Templates de Blocos

```
base-pcb/templates/
├── rp2350-minimal/         # RP2350 + cristal + flash + USB-C + SWD
├── cortex-a55-ddr5/        # Cortex-A55 + DDR5 SODIMM
├── ethernet-rgmii/         # Ethernet PHY + magnetics + RJ45
├── usb-c-pd/               # USB-C + PD controller
├── power-3v3/              # Buck 5V→3.3V @ 1A + LDO + decoupling
├── power-1v8/              # Buck 3.3V→1.8V @ 500mA
├── power-1v1/              # Buck 3.3V→1.1V @ 2A (core)
├── swd-jtag/               # 10-pin Cortex debug header
└── i2c-header/             # 4-pin I2C header + pull-ups
```

Cada template contém:
- `template.kicad_sch` — esquemático
- `template.kicad_pcb` — footprint + posicionamento
- `template.yaml` — metadados (pinos de conexão, parâmetros)

## Geração Programática do Esquemático

```rust
fn generate_kicad_sch(components: &[PlacedComponent], nets: &[Net]) -> String {
    let mut sch = String::new();
    sch.push_str("(kicad_sch (version 20231121) (generator \"base-pcb\")\n");
    sch.push_str("  (title_block\n");
    sch.push_str("    (title \"B.A.S.E. Generated\")\n");
    sch.push_str(&format!("    (date \"{}\")\n", chrono::Local::now().format("%Y-%m-%d")));
    sch.push_str("  )\n");
    
    for comp in components {
        sch.push_str(&comp.to_sexpr());
    }
    for net in nets {
        sch.push_str(&net.to_sexpr());
    }
    
    sch.push_str(")\n");
    sch
}
```

## Validação com kicad-cli

```bash
# Verificar DRC do esquemático
kicad-cli sch export netlist project.kicad_sch --output project.net

# Verificar DRC da PCB
kicad-cli pcb drc project.kicad_pcb

# Gerar gerber
kicad-cli pcb export gerber project.kicad_pcb --output gerber/
```
