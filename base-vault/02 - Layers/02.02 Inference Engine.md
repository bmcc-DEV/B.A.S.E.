# Behavioral Inference Engine

## Visão Geral

Coração do B.A.S.E. Responsável por transformar observações brutas (acessos MMIO, traces, FSM do firmware) em um modelo comportamental estruturado.

## Arquitetura em 4 Estágios

```
MMIO Accesses + Traces + Drivers
         │
         ▼
┌─────────────────────────────────┐
│ Stage 1: Block Extraction       │
│ Clustering por endereço base    │
│ Classificação do tipo de bloco  │
│ Separação register file / FIFO  │
│ / doorbell / status / DMA desc  │
└──────────────┬──────────────────┘
               │
               ▼
┌─────────────────────────────────┐
│ Stage 2: Protocol Inference     │
│ N-gram analysis de sequências   │
│ Detecção de padrões de escrita  │
│ Identificação de register       │
│ purpose por offset + padrão     │
└──────────────┬──────────────────┘
               │
               ▼
┌─────────────────────────────────┐
│ Stage 3: FSM Extraction         │
│ Graph mining sobre estados      │
│ Inferência de transições        │
│ Matriz de latência entre estados│
└──────────────┬──────────────────┘
               │
               ▼
┌─────────────────────────────────┐
│ Stage 4: Specification Gen      │
│ Converte em HardwareSpec YAML   │
│ Calcula confidence global       │
│ Relatório de inferência         │
└─────────────────────────────────┘
```

## Stage 1 — Block Extraction

```rust
struct BlockCluster {
    base_address: u64,
    size: u64,
    block_type: BlockType,
    registers: Vec<RawRegister>,
    confidence: f64,
}

enum BlockType {
    RegisterFile,
    Fifo,
    Doorbell,
    Status,
    DmaDescriptor,
    Unknown,
}

fn extract_blocks(accesses: &[MmioAccess]) -> Vec<BlockCluster> {
    let groups = group_by_page(accesses);
    
    groups.into_iter().map(|group| {
        let write_ratio = group.writes() / group.total() as f64;
        let read_ratio = group.reads() / group.total() as f64;
        let access_pattern = detect_pattern(&group);
        let block_type = classify_block(write_ratio, read_ratio, &access_pattern);
        
        BlockCluster {
            base_address: group.base,
            size: group.size,
            block_type,
            registers: extract_registers(&group),
            confidence: calculate_confidence(&group),
        }
    }).collect()
}

fn classify_block(wr: f64, rd: f64, pattern: &AccessPattern) -> BlockType {
    match (wr, rd, pattern) {
        (w, r, _) if w > 0.9 && r < 0.1 && has_few_unique_values(pattern) => BlockType::Doorbell,
        (w, r, _) if w < 0.1 && r > 0.9 && is_periodic(pattern) => BlockType::Status,
        (w, r, _) if w > 0.4 && w < 0.6 && has_fixed_offsets(pattern) => BlockType::RegisterFile,
        (w, r, _) if (w > 0.7 || r > 0.7) && is_sequential(pattern) => BlockType::Fifo,
        (_, _, p) if is_large_block(p) => BlockType::DmaDescriptor,
        _ => BlockType::Unknown,
    }
}
```

## Stage 2 — Protocol Inference

```rust
struct InferredProtocol {
    registers: Vec<RegisterRole>,
    sequences: Vec<AccessSequence>,
    timing: ProtocolTiming,
}

enum RegisterRole {
    Control,
    Status,
    AddressPtr,
    DataLength,
    DataPort,
    Trigger,
    InterruptMask,
    InterruptAck,
    ClockConfig,
    Unknown,
}

fn infer_protocol(block: &BlockCluster) -> InferredProtocol {
    let ngrams = collect_ngrams(&block.accesses, 4);
    let sequences = extract_frequent_sequences(&ngrams, 0.1);
    
    InferredProtocol {
        registers: sequences.iter().flat_map(|seq| infer_roles(seq)).collect(),
        sequences,
        timing: measure_timing(&block.accesses),
    }
}
```

## Stage 3 — FSM Extraction

```rust
struct InferredFSM {
    states: Vec<State>,
    transitions: Vec<Transition>,
    entry_state: StateId,
    exit_states: Vec<StateId>,
    latency_matrix: HashMap<(StateId, StateId), LatencyRange>,
}

fn extract_fsm(block: &BlockCluster, protocol: &InferredProtocol) -> InferredFSM {
    let state_values = find_state_defining_registers(block);
    let transitions = find_transitions(block, &state_values);
    let clean_transitions = filter_noise(&transitions, 0.05);
    
    InferredFSM {
        states: state_values.iter()
            .map(|(name, val)| State { name: name.clone(), value: *val })
            .collect(),
        transitions: clean_transitions,
        entry_state: find_entry_state(&clean_transitions),
        exit_states: find_exit_states(&clean_transitions, block),
        latency_matrix: calculate_latencies(block, &clean_transitions),
    }
}
```
