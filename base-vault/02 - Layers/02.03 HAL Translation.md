# HAL de Tradução MMIO

## Problema

O firmware original foi escrito para endereços MMIO específicos do ASIC original. No novo hardware, esses endereços não existem mais. Precisamos de uma **camada de tradução**.

## Estratégias

| Estratégia | Overhead | Compatibilidade | Complexidade |
|-----------|----------|----------------|--------------|
| **MMU-based** | 0 (HW) ~10 ciclos (TLB miss) | Alta (transparente) | Média |
| **Software trap** | ~100ns por acesso | Média (precisa de handler) | Alta |
| **PIO Proxy** | ~20ns (RP2350) | Baixa (32 bits, 2 SMs) | Média |
| **Stub Driver** | ~10ns (função inline) | Baixa (precisa substituir driver) | Baixa |

## Abordagem Híbrida Recomendada

```
         ┌─────────────────────────────────────────┐
         │           Firmware Original              │
         │    (acessa 0x10000000, 0x10010000...)     │
         └────────────────┬────────────────────────┘
                          │
                          ▼
         ┌─────────────────────────────────────────┐
         │      MMU Translation Table (HW)          │
         │  0x10000000 → 0x20000000 (DMA + GPIO)    │
         │  0x10010000 → Trap Handler               │
         └────────────────┬────────────────────────┘
                          │
                ┌─────────┴──────────┐
                ▼                    ▼
     ┌──────────────────┐  ┌──────────────────┐
     │ DMA Controller   │  │ Software Handler │
     │ (hardware, zero  │  │ PIO emulation    │
     │  overhead)        │  │ ~100ns penalty   │
     └──────────────────┘  └──────────────────┘
```

## Geração Automática

```rust
struct HalGenerator {
    spec: HardwareSpec,
    target: TargetPlatform,
}

impl HalGenerator {
    fn generate(&self) -> GeneratedHal {
        let mut hal = GeneratedHal::new();
        
        for block in &self.spec.blocks {
            let mapping = self.map_block(block);
            let code = match mapping.strategy {
                TranslationStrategy::MmuPage(addr) => self.gen_mmu_entry(block.base_address, addr),
                TranslationStrategy::TrapHandler => self.gen_trap_handler(block),
                TranslationStrategy::PioEmulation(pio_cfg) => self.gen_pio_program(pio_cfg),
            };
            hal.add_translation(block.id, code);
        }
        
        hal.add_interrupt_table(self.gen_irq_table());
        hal.add_timing_compensation(self.gen_timing_delays());
        hal.add_init_sequence(self.gen_init_sequence());
        hal
    }
    
    fn map_block(&self, block: &FunctionalBlock) -> BlockMapping {
        match block.kind {
            BlockKind::Dma => BlockMapping {
                strategy: TranslationStrategy::MmuPage(self.target.dma_address()),
                requires_sw_handler: false,
            },
            BlockKind::Gpu => BlockMapping {
                strategy: TranslationStrategy::TrapHandler,
                requires_sw_handler: true,
            },
            BlockKind::I2c => BlockMapping {
                strategy: TranslationStrategy::PioEmulation(self.target.pio_for_i2c()),
                requires_sw_handler: false,
            },
            _ => BlockMapping {
                strategy: TranslationStrategy::SoftwareStub,
                requires_sw_handler: true,
            },
        }
    }
}
```

## Timing Compensation

```c
// Gerado automaticamente pelo B.A.S.E.
// Arquivo: hal/timing_compensation.c

static inline void timing_compensate(enum BlockId block, enum Operation op) {
    switch (block) {
        case BLOCK_GPU:
            switch (op) {
                case OP_WAKE:
                    delay_us(5);  // Original: 3us, Novo: 0.5us
                    break;
                case OP_TRANSFER:
                    break;
            }
            break;
            
        case BLOCK_AUDIO:
            hw_timer_delay(AUDIO_SAMPLE_RATE_DELAY);
            break;
    }
}
```

## IRQ Translation

```rust
struct IrqTranslator {
    original_vectors: HashMap<u8, IrqSpec>,
    target_irqs: HashMap<BlockId, u8>,
}

impl IrqTranslator {
    fn translate_vector(&self, original: u8) -> Option<(u8, bool)> {
        let spec = self.original_vectors.get(&original)?;
        let target_irq = self.target_irqs.get(&spec.owner)?;
        Some((*target_irq, spec.ack_type == AckType::Write))
    }
}
```

## Matriz de Decisão

| Critério | MMU | Trap | PIO | Stub Driver |
|----------|-----|------|-----|-------------|
| Requer MMU | ✅ | ❌ | ❌ | ❌ |
| Overhead | 0 | ~100ns | ~20ns | ~10ns |
| Compatibilidade | Total | Total | Parcial | Baixa |
| Complexidade | Média | Alta | Média | Baixa |
| Timing exato | ✅ | ❌ | ✅ | ❌ |
