# QMP savevm (F2)

```bash
# Sobe QEMU + status + probe-savevm + quit (um comando)
./target/debug/base virt demo qmp

# Cliente contra guest já a correr (socket default /tmp/base-qmp.sock)
# Nota: `demo qmp` sobe e faz quit — para estes comandos deixa QEMU vivo noutro terminal.
./target/debug/base virt qmp status
./target/debug/base virt qmp savevm --tag base_snap
./target/debug/base virt qmp loadvm --tag base_snap
./target/debug/base virt qmp probe-savevm
```

`savevm` depende da máquina QEMU (sem disco de vmstate falha — esperado e reportado).
