# BIR Twin (F1)

```bash
./target/debug/base virt bir-twin \
  --spec examples/pilot_moto_g35/virt/hardware_spec_mame_stub.yaml \
  --evidence /tmp/ing_mame/evidence_db.yaml \
  -o /tmp/bir_twin
```

Emite `bir_device.yaml` + `bir_twin_report.json` (regs após replay).

# QMP savevm (F2)

```bash
./target/debug/base virt qmp --socket /tmp/virt/qmp.sock savevm --tag base_snap
./target/debug/base virt qmp --socket /tmp/virt/qmp.sock loadvm --tag base_snap
./target/debug/base virt qmp --socket /tmp/virt/qmp.sock probe-savevm --tag base_snap
```

Depende da máquina QEMU (muitas configs recusam snapshot) — erro QMP é esperado e reportado.
