# Continuous watch (F3)

```bash
# Offline (ficheiro completo em ticks)
./target/debug/base virt watch \
  --spec examples/pilot_moto_g35/virt/hardware_spec_mame_stub.yaml \
  --trace examples/pilot_moto_g35/virt/sample_mmio.ndjson \
  --window-events 2 -o /tmp/watch

# Live poll (plugin a escrever outfile)
./target/debug/base virt watch \
  --spec …/hardware_spec.yaml \
  --trace /tmp/plugin_trace.ndjson \
  --poll-ms 200 --poll-timeout-sec 10 -o /tmp/watch_live
```

Emite `continuous_diff.json` com série `ticks[]` (hit_rate + Ψ).
