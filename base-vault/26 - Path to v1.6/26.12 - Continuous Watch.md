# Continuous watch (F3)

```bash
./target/debug/base virt demo watch
# → /tmp/base_virt_demo/watch/continuous_diff.json

# Ou script:
./examples/pilot_moto_g35/demo_virt.sh watch
```

Poll live (plugin a escrever NDJSON) continua no comando longo:

```bash
./target/debug/base virt watch \
  --spec examples/pilot_moto_g35/virt/hardware_spec_mame_stub.yaml \
  --trace /tmp/plugin_trace.ndjson \
  --poll-ms 200 --poll-timeout-sec 10 -o /tmp/watch_live
```
