# HIL Lab Gate A — operacional

```bash
# Default (A1/A2 BLOCK no CI)
base hil lab-status --sop examples/hil_lab/SOP.md -o out/

# Fechar A1/A2 (rehearsal)
cargo build -p base-cli --features hil_programmer
export BASE_HIL_ALLOW_FLASH=1
export BASE_HIL_PROGRAMMER_CMD='test -f {image}'
base hil lab-status --sop examples/hil_lab/SOP.md --mock-detected -o out/
# → A1/A2 GREEN; production=false; A5 aberto → lab_assist_ready=false

./examples/hil_lab/run_hil_lab_assist.sh
```

| Id | Software hoje | Lab Cliente |
|----|---------------|-------------|
| A1 | `--mock-detected` / `BASE_HIL_MOCK_DETECTED` / `hil_usb` | probe USB Detected estável |
| A2 | `--features hil_programmer` + ALLOW_FLASH + CMD | CMD do operador documentado |
| A3 | ficheiro SOP | SOP adaptado |
| A4 | invariante código | auditoria receipts |
| A5 | `--sow-signed` | contrato assinado |

`lab_assist_ready=true` só com todos verdes (incl. A5). Ainda **≠** production turnkey.  
`--mock-detected` ≠ USB real — só rehearsal offline.
