## Estrutura do Grafo

```text
       ┌──────────┐
       │   CPU    │
       └────┬─────┘
            │ controls
            ▼
       ┌──────────┐  has_register  ┌──────────┐
       │   GPU    │───────────────►│ CONTROL  │
       └────┬─────┘                │ value: 0 │
            │                      │ value: 1 │
            │ triggers             │ value: 2 │
            ▼                      └──────────┘
       ┌──────────┐  has_timing  ┌─────────────┐
       │DMA_START │─────────────►│ 100ns..400ns │
       └──────────┘              └─────────────┘
```

## Schema do Grafo

```rust
enum NodeType {
    SoC,
    Block(BlockKind),
    Register,
    Event,
    Interrupt,
    Timing,
    Value,
    Function,
    Driver,
}

enum EdgeType {
    Contains,       // SoC → Block
    Controls,       // Block → Register
    Triggers,       // Event → Interrupt
    TransitionsTo,  // State → State
    HasTiming,      // Event → Timing
    DependsOn,      // Block → Block
    MapsTo,         // Original → New (evolution)
}
```
