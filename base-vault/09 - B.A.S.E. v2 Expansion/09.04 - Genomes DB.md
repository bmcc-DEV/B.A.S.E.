# Fase 4 — Hardware Genome Database

**Esforço:** Contínuo | **Risco:** Baixo | **Inovação:** ★★★

> Base de conhecimento massiva de componentes, cada um com seu "genoma" padronizado.

## Formato Genome v2 (Estendido)

```yaml
genome:
  part: RP2350
  version: 2
  supersedes: RP2040

  # Identificação
  manufacturer: Raspberry Pi
  category: mcu
  package: QFN-56

  # Barramentos
  bus:
    - type: QSPI
      speed: 133MHz
    - type: PIO
      lanes: 8

  # Periféricos (compatível com block inference)
  peripherals:
    - name: SPI0
      compatible: spi
      max_speed: 50MHz
    - name: I2C0
      compatible: i2c
      max_speed: 1MHz
    - name: DMA
      channels: 8
      chainable: true

  # Timing para o Constraint Solver
  timing:
    cpu_fetch:   8ns
    dma_setup:   50ns
    dma_xfer:    12ns per_byte
    gpio_toggle: 5ns

  # Potência
  power:
    active:    90mW
    sleep:     5mW
    deep_sleep: 0.5mW

  # Preço e disponibilidade
  pricing:
    unit_1k: 1.50
    unit_10k: 1.20
    distributor: [DigiKey, Mouser]
    stock: high

  # Compatibilidade
  compatible_with: [RP2040]  # drop-in replacements
```

O B.A.S.E. v2 pode buscar atualizações:

```bash
base genome update                # puxa genomas mais recentes
base genome search "spi, dma, <$1"  # busca componentes
base genome diff rp2350 rp2040     # diferenças entre gerações
```
