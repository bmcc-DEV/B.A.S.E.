# Fase 9 — Closed Feedback Loop

**Esforço:** 4 semanas | **Risco:** Médio | **Inovação:** ★★★

> Pipeline recursivo, não linear. Cada erro melhora o modelo. O ciclo nunca termina.

## Conceito

```text
Analyze → Model → Synthesize → Validate → Error → Refine → Analyze
   │                                                       │
   └────────────────────── LOOP ◄──────────────────────────┘
```

## Diagnóstico de Erros

```rust
enum ErrorClass {
    MissingRegister { offset: u32 },
    WrongTiming { event: String, expected: Latency, actual: u64 },
    UnmappedAddress { address: u64 },
    MissingInterrupt { vector: u8 },
    ExtraEvent { description: String },
}
```

## Refinamento Automático

```rust
fn refine_model(model: &mut BirDevice, errors: &[DiagnosedError]) {
    for error in errors {
        match error {
            ErrorClass::MissingRegister { offset } => {
                // Adiciona registro desconhecido ao modelo
                model.registers.push(BirRegister {
                    offset: *offset,
                    name: format!("unknown_{:x}", offset),
                    confidence: 0.3, // baixa confiança, refinável
                    ..default()
                });
            }
            ErrorClass::WrongTiming { event, .. } => {
                // Relaxa contrato temporal
                if let Some(contract) = model.contract_mut(event) {
                    contract.latency.max *= 2; // dobra tolerância
                }
            }
            _ => {}
        }
    }
}
```
