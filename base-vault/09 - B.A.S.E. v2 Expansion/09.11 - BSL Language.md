## Gramática BSL (PEG)

```pest
device   = { "device" ~ name ~ "@"? ~ address ~ "{" ~ body ~ "}" }
name     = { (ASCII_ALPHA | "_") ~ (ASCII_ALPHANUMERIC | "_")* }
address  = { "0x" ~ (ASCII_HEX_DIGIT)+ }

body     = { (register_decl | event_decl | interrupt_decl | timing_decl | contract_decl)* }

register_decl = { name ~ "@" ~ offset ~ ":" ~ access ~ ("=" ~ value)? ~ ";" }
offset        = { "0x" ~ (ASCII_HEX_DIGIT)+ }
access        = { "rw" | "ro" | "wo" }

event_decl    = { name ~ ":" ~ trigger ~ ";" }
trigger       = { (access | read | write) ~ name ~ "[" ~ range ~ "]" ~ "=" ~ value }
```

## Exemplo BSL Completo

```bsl
// B.A.S.E. Behavior Specification Language
// Exemplo: GPU com DMA

device GPU @ 0x10000000 {
    registers {
        CONTROL   @ 0x00: rw = 0x00000000;
        STATUS    @ 0x04: ro;
        BUF_ADDR  @ 0x08: wo;
        LENGTH    @ 0x0C: rw;
        IRQ_MASK  @ 0x10: rw = 0x00000000;
        IRQ_STATUS @ 0x14: ro;
    }

    events {
        DMA_START:   write CONTROL[0] = 1;
        DMA_COMPLETE: read STATUS[7]  = 1;
        IRQ_FIRE:    read IRQ_STATUS != 0;
    }

    interrupts {
        IRQ_GPU:  level high;
        IRQ_DMA:  level high;
    }

    timing {
        dma_setup:  100ns..400ns;
        dma_xfer:   50ns..200ns per_word;
        irq_latency: 50ns..150ns;
    }

    contract {
        must_occur_before: DMA_COMPLETE -> IRQ_GPU;
        window: 10us;
    }
}
```

## LSP Server

O Language Server Protocol permite:

- **Completions**: auto-completar nomes de registradores
- **Diagnostics**: erro se registrador duplicado, referência inválida
- **Hover**: mostrar documentação do registrador
- **Go to definition**: navegar entre referências
- **Code actions**: extrair bloco, renomear

```bash
base bsl lsp                            # Inicia LSP server
# VS Code: instalar extensão BSL
# Neovim: :LspStart bsl
```

```bash
base bsl compile device.bsl             # BSL → BIR (YAML)
base bsl validate device.bsl            # Verifica erros
base bsl fmt device.bsl                 # Formata código
base bsl doc device.bsl                 # Gera documentação HTML
```
