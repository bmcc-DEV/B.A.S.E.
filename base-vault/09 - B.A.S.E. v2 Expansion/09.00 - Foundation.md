## Estrutura de Crates

```
base-bir/          ← Behavioral IR: AST + validador + serializador
  src/
    bir.rs         ← Tipos do IR
    validate.rs    ← Verificador
    serialize.rs   ← YAML bridge
    mod.rs

base-bsl/          ← Behavior Specification Language
  src/
    grammar.pest   ← Gramática PEG
    parser.rs      ← BSL → BIR compiler
    mod.rs
```
