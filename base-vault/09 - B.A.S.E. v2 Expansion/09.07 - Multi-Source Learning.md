## Confiança por Fonte

| Fonte | Confiança | Exemplo |
|-------|-----------|---------|
| Firmware (execução real) | 0.8 | MMIO acessos observados |
| Driver Linux (código) | 0.6 | `struct resource` + `ioremap` |
| Datasheet oficial | 0.6 | Tabela de registradores |
| Trace Saleae | 0.9 | Captura direta do barramento |
| DTB (Device Tree) | 0.5 | `reg = <0x10000000 0x1000>` |
| Schema elétrico PDF | 0.3 | Extração por OCR |
| LLM (inferência) | 0.2 | "Este parece ser um registrador de status" |

```rust
struct Evidence {
    source: SourceType,
    confidence: f64,
    bir_fragment: BirDevice,
}

fn fuse_evidence(sources: Vec<Evidence>) -> BirDevice {
    let mut merged = BirDevice::empty();

    for evidence in sources {
        for reg in evidence.bir_fragment.registers {
            let weight = evidence.confidence;
            merged.upsert_register(reg, weight);
        }
        // Mesmo para events, interrupts, contracts...
    }

    // Detecta contradições
    for contradiction in detect_conflicts(&merged) {
        tracing::warn!("Contradiction: {:?}", contradiction);
        // Resolve: maior confiança vence
    }

    merged
}
```

## Ingestão de Datasheet PDF

```rust
fn extract_mmio_from_pdf(path: &Path) -> Result<BirDevice> {
    let text = pdf_extract::extract_text(path)?;

    // Procura por tabelas de registradores
    // Padrão típico: "0x00" "CONTROL" "R/W" "0x00000000"
    let re = Regex::new(r"(0x[0-9A-Fa-f]+)\s+(\w+)\s+(R/W|RO|WO)")?;

    for cap in re.captures_iter(&text) {
        let offset = u32::from_str_radix(&cap[1], 16)?;
        let name = cap[2].to_string();
        let access = parse_access(&cap[3])?;

        registers.push(BirRegister { offset, name, access, .. });
    }

    BirDevice { registers, .. }
}
```
