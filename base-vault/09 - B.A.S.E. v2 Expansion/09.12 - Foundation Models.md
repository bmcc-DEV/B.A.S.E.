## Classificador de Blocos

```rust
use tract_onnx::prelude::*;

struct BlockClassifier {
    model: SimplePlan<TypedFact, Box<dyn TypedOp>, Graph<TypedFact, Box<dyn TypedOp>>>,
}

impl BlockClassifier {
    fn new() -> Self {
        let model = onnx()
            .model_for_path("models/block_classifier.onnx")
            .unwrap()
            .into_optimized()
            .unwrap()
            .into_runnable()
            .unwrap();

        Self { model }
    }

    fn predict(&self, features: &[f32]) -> HashMap<String, f32> {
        let tensor = tensor2(features);
        let result = self.model.run(tvec!(tensor.into())).unwrap();

        // Resultado: probabilidades por classe
        // { "Doorbell": 0.93, "RegisterFile": 0.04, ... }
        let output = result[0].to_array_view::<f32>().unwrap();
        // ...
    }
}
```

## Modelos Específicos

| Modelo | Arquitetura | Input | Output | Precisão Alvo |
|--------|------------|-------|--------|---------------|
| Block Classifier | RandomForest (→ ONNX) | 8 features de acesso | BlockKind | > 90% |
| Timing Predictor | Regressão linear (→ ONNX) | BlockKind + clock | ns por operação | ±20% |
| Protocol Detector | CNN 1D | Sequência de offsets | Doorbell/FIFO/DMA | > 85% |
| Register Namer | Bi-LSTM + Attention | Offset + valores | Nome do registrador | > 70% |

## Pipeline de ML Integrado

```rust
fn enhanced_extract_blocks(accesses: &[MmioAccess]) -> Vec<BlockCluster> {
    let mut clusters = extraction::extract_blocks(accesses);

    // Estágio 1 tradicional + ML
    let classifier = BlockClassifier::new();
    for cluster in &mut clusters {
        let features = extract_features(cluster);
        let probs = classifier.predict(&features);

        // ML sugere tipo, heurística vota
        cluster.block_type = fuse_classification(
            cluster.block_type,  // heurística
            probs,               // ML
        );
    }

    clusters
}
```
