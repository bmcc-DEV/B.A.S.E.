# Fase 6 — HIL Cluster

**Esforço:** 8 semanas | **Risco:** Alto | **Inovação:** ★★★★★

> Rede de captura multi-protocolo para hardware-in-the-loop.

## Conceito

Um probe de hardware dedicado (RP2350) que se conecta aos barramentos do hardware original, captura ciclos de leitura/escrita com timestamp preciso, e envia para o `base-check`.

## Tarefas

- [ ] Firmware RP2350 probe core (Rust, rp-hal)
- [ ] Protocolo de captura: UART, SPI, I2C, GPIO
- [ ] Sincronização temporal entre múltiplos probes (PIO sync)
- [ ] Host agent (Rust, `probe-rs` para comunicação)
- [ ] Integração com `base-check`: HIL mode recebe stream de eventos
- [ ] Suporte a barramento paralelo 8/16/32 bits via PIO
- [ ] Documentação de montagem: BOM do probe + PCB

## Firmware RP2350 (Rust)

```rust
// HIL Probe Core — captura de barramento paralelo via PIO
#![no_std]
#![no_main]

use rp235x_hal as hal;

#[hal::entry]
fn main() -> ! {
    let mut pac = hal::pac::Peripherals::take().unwrap();
    let mut pio = pac.PIO0;

    // Configura PIO para capturar barramento paralelo de 8 bits
    let sm = 0;
    pio.sm_exec(sm, &[
        pio!({pull block}),           // Aguarda trigger do host
        pio!({in y, 32}),             // Lê 32 bits do barramento
        pio!({push block}),           // Envia pela FIFO
    ]);

    loop {
        if let Some(sample) = pio.sm_pull(sm) {
            let timestamp = timer.get_counter();
            hil_send_sample(timestamp, sample); // via USB
        }
    }
}
```

## Host Agent

```rust
// hil-agent: recebe stream do probe e alimenta base-check
fn capture_loop() -> Result<()> {
    let mut probe = HilProbe::connect("usb:1234:5678")?;
    let mut trace = DeviceTrace::new("HIL capture");

    loop {
        let sample = probe.read_sample()?;
        trace.events.push(TraceEvent {
            timestamp_ns: sample.timestamp,
            channel: format!("BUS_{}", sample.channel),
            event_type: EventType::MmioWrite,
            address: sample.address,
            value: Some(sample.data),
        });
    }
}
```
