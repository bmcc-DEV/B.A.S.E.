# Tools Reference

## Ferramentas de Análise

| Ferramenta | Uso no Pipeline | Camada |
|-----------|----------------|--------|
| **Ghidra** | Reverse engineering complementar (SRE) | 1-2 |
| **Radare2** | Disassembly + analysis | 1-2 |
| **Binwalk** | Extração de firmware de imagens | 1 |
| **Wireshark** | Captura e análise de tráfego USB/PCI/I²C | 1 |
| **Saleae Logic** | Captura de sinais digitais (SPI, I²C, parallel) | 1 |
| **Capstone** | Disassembly ARM64/ARM32/Thumb | 2 |
| **kicad-cli** | Validação e export de PCBs | 6a |
