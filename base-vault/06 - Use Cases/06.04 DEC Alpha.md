# Use Case: DEC Alpha

## Sistema Original

| Componente | Especificação |
|------------|--------------|
| CPU | DEC Alpha 21164 @ 500MHz |
| RAM | ECC DDR ? |
| Chipset | DECchip 21174 Core Logic |
| Storage | SCSI / IDE |
| Rede | DEC DE500 (Tulip) |
| Firmware | AlphaBIOS / SRM |
| OS | OpenVMS, Tru64 UNIX, Linux |

## Análise

Complexidade: **Alta** (arquitetura Alpha bizarra: little-endian incomum para época, sem byte addressing)

## Por que interessante

1. **Arquitetura exótica**: Alpha é radicalmente diferente de ARM/x86
2. **Documentação Pública**: Manual da arquitetura disponível
3. **Validação de inference engine**: MMIO mapping de chipset Core Logic
4. **BIOS shim**: Tradução AlphaBIOS → UEFI stub

## Alvo B.A.S.E. (Teórico)

| Função | Abordagem |
|--------|-----------|
| CPU | Cortex-A55 (tradução via software) |
| RAM | DDR3 (8GB) |
| Storage | SD Card (SPI) |
| Rede | W5500 Ethernet |
| Firmware | UEFI stub com tradução de serviços AlphaBIOS |

## Status

**Experimental.** Útil para validar inference engine com arquitetura não-ARM64.
