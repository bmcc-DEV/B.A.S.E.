### Passos

1. **Seed** вҖ” `HardwareSpec` (+ device QEMU Camada 9 ou `-machine virt` + firmware).
2. **Instrument** вҖ” plugin/device вҶ’ eventos `MmioRead/Write`, `Irq` (schema `EvidenceDb`).
3. **Observe** вҖ” janela temporal вҶ’ batch de evidГӘncia.
4. **Score** вҖ” `ОЁ(Evidence, Spec)`; conclusiveness match / inconclusive / no-match.
5. **Refine** вҖ” ajustar hipГіtese / regenerar device / promover padrГЈo paleo.
6. **Probe** (fase 3) вҖ” QMP stop/cont/savevm/inject IRQ.
7. **Halt** вҖ” estagnaГ§ГЈo / limiar; flags honesty sempre false.

## Formato de trace (MVP)

NDJSON, uma linha por evento:

```json
{"op":"mmio_write","addr":"0x10000000","value":"0x1","ts_ns":1000}
{"op":"mmio_read","addr":"0x10000004","ts_ns":2000}
{"op":"irq","vector":32,"polarity":"rising","ts_ns":3000}
```

Parser: `base_virt::trace::parse_ndjson_line` вҶ’ `EvidenceEntry`.
