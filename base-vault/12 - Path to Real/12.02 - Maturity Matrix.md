# 📊 Maturity Matrix

Legenda: **REAL** · **PARCIAL** · **SCAFFOLD** · **QUEBRADO**

Atualizar após cada sprint. Baseline = pós Path to v1.1 **A4** (2026-07-16). Tag estável: `v1.0.0` · path: [[21 - Path to v1.1/21.00 - Index|Path to v1.1]].
