# Arquiteturas suportadas — Analyze / SpecterProbe

Documentado em R1 ([[12.11 - Sprint R1 Analyze]]).

| Path | Arch | Status |
|------|------|--------|
| Capstone via SpecterProbe `lift_binary` | **AArch64 (ARM64)** | Principal |
| Heuristic word scan (`is_likely_mmio`) | Qualquer blob | Fallback com `WARN` |
| `--mmio-traces` JSON/YAML | Independente de ISA | **Preferido** no piloto |

## Headers stripped

| Magic | Ação |
|-------|------|
| `DHTB` | Skip 0x200 |
| MTK `A0 0A 1A 00` | Skip 32 |
| `ANDROID!` | Skip ~1648 (boot image simplificado) |
| gzip `1F 8B` | Sem decompress — passa adiante |

## Regras

1. Sem Capstone hits → heuristic + `WARN` + contagem
2. Sem candidatos → Evidence vazia, **não crash**
3. Piloto usa `--mmio-traces` como ground truth
