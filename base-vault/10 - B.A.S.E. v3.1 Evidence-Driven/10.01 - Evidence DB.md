# Evidence DB — Fatos Puros

> *"Eu observei estes comportamentos" — nunca "isto é uma GPU".*

## Formato on-disk (CLI — 2026-07 / R1)

Emitido por `base analyze` como `evidence_db.yaml`:

```yaml
source: examples/pilot/fw.bin
created_at: 1721080000
entries:
  - id: ev_0000
    evidence_type:
      type: MmioWrite
      address: 0x40034000
      value: 1
    context:
      function: uart_init
      instr: "0x0"
  - id: ev_0001
    evidence_type:
      type: MmioRead
      address: 0x40034004
    context:
      function: uart_status
      instr: "0x4"
```

Schema de entrada `--mmio-traces`: `examples/pilot/expected/mmio_traces.schema.md`

## Formato conceitual (vault)

```yaml
# evidence.yaml — SOMENTE fatos observados
evidence:
  - id: ev_001
    type: mmio_write
    address: 0x10000000
    value: 0x01
    context:
      function: init_gpu
      instruction_addr: 0x1004
```

> A serialização serde usa `evidence_type: { type: MmioWrite, ... }` (tag interna). Não misturar assessment/confidence aqui.
## Tipos de Evidência

| Tipo | Descrição | Campos |
|------|-----------|--------|
| `mmio_write` | Escrita em registrador mapeado | address, value |
| `mmio_read` | Leitura de registrador mapeado | address |
| `irq` | Interrupção detectada | vector, polarity |
| `dma` | Transferência DMA | source, destination, size |
| `gpio_toggle` | GPIO alternado | pin, value |
| `spi_transfer` | Transferência SPI | mosi, miso, cs |
| `i2c_transfer` | Transferência I2C | address, data |
| `function_call` | Chamada de função detectada | from, to |

## Regras

1. **Nenhum confidence score** na evidência
2. **Nenhuma interpretação** (não dizer "isto é DMA controller")
3. **Imutável** — uma vez registrada, não muda
4. **Versionável** — cada execução gera um novo conjunto

## Implementação (Rust)

```rust
struct EvidenceDb {
    entries: Vec<EvidenceEntry>,
    source: String,
    created_at: u64,
}

enum EvidenceType {
    MmioWrite { address: u64, value: Option<u64> },
    MmioRead { address: u64 },
    Irq { vector: u8, polarity: IrqPolarity },
    Dma { source: u64, dest: u64, size: Option<u64> },
    GpioToggle { pin: u8, value: bool },
}

struct EvidenceEntry {
    id: String,
    type: EvidenceType,
    context: HashMap<String, String>,
}
```
