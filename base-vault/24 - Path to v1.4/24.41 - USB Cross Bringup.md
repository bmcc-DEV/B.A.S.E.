# G35 — USB × DTB + bring-up

## Alvo de port (escolhido)

**`linux_wedge_uart_ufs_g35`** — early bring-up AArch64 no ums9620 (UART → arch timer/GIC → UFS).  
GPU/display/modem = depois. ≠ TaurOS turnkey · `generates_os: false`.

## Comandos

```bash
./examples/pilot_moto_g35/run_usb_probe.sh
./examples/pilot_moto_g35/run_usb_cross.sh
# → out_real/usb_cross/BRINGUP_CHECKLIST.md
```

Ou:

```bash
base port usb-cross \
  --usb examples/pilot_moto_g35/out_real/usb_probe/usb_hw_inventory.yaml \
  --platform examples/pilot_moto_g35/out_real/platform_vendor_boot/platform_inventory.yaml \
  -o examples/pilot_moto_g35/out_real/usb_cross
```

## Ordem P0→P2

1. UART (`20200000.serial` …)  
2. Arch timer + GIC (DTB)  
3. UFS (`22000000.ufs`)  
4. GPIO/pinctrl  
5. GPU/DPU / USB3 (opcional)

Cruzar sempre com `port_package_lk/` (fósseis / must rewrite).
