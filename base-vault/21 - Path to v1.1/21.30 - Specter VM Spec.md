# Specter VM — spec (L28–L30)

Máquina Forth-like + política Lua para **estudo autónomo** de comportamento.

## Palavras

| Word | Efeito |
|------|--------|
| `OBSERVE` | Analisa erros estruturais no spec atual |
| `SCORE` | Empilha pass_rate (×1000 como int) |
| `REFINE` | Uma iteração de `FeedbackLoop` |
| `STAGNATE?` | Empilha 1 se última iteração sem mudanças |
| `CONVERGED?` | Empilha 1 se pass_rate ≥ threshold |
| `HALT` | Para o loop |

## Lua policy

```lua
return {
  threshold = 0.9,
  max_steps = 32,
  continuous = false,
}
```

## Report

`study_report.json`: `steps`, `stop_reason`, `stagnated`, `auto_fix_complete: false`, `final_pass_rate`.
