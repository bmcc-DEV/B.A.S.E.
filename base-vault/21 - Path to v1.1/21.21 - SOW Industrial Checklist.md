# SOW checklist — industrial (v1.1)

> Copiar para o contrato. **Não** SaaS turnkey. Humano no loop.
> Base: [[20 - Path to v1.0/20.21 - SOW Industrial Checklist|SOW v1.0]].

## Escopo

- [ ] Analyze + Evidence (RP e/ou STM32)
- [ ] Specter `base study` (Forth+Lua) se acordado — report com `stop_reason`
- [ ] Goldens `diff` / reconstruct / evolve métricas / hil host / pcb draft
- [ ] Relatório de viabilidade + handoff

## Não-escopo

- [ ] PCB fabricável / HIL production / auto-fix / SaaS turnkey / ASIC drop-in

## Aceite

1. Gates RP verdes.  
2. Se study: `run_study.sh` ou bloqueio honesto.  
3. Relatório sem claims Não-escopo.
