# SOW checklist — industrial (v1.0)

> Copiar para o contrato. **Não** SaaS turnkey. Humano no loop.
> Base: [[19 - Path to v0.9/19.21 - SOW Industrial Checklist|SOW v0.9]].
> **v1.0 ≠** produto industrial completo.

## Escopo (checklist)

- [ ] Analyze + Evidence (RP e/ou STM32)
- [ ] Capstone/disasm no wedge STM32 se acordado (blob sintético ≠ Thumb silício)
- [ ] Dual USART+SPI2 e/ou USART+I2C1 e/ou **triple** USART+SPI2+I2C1 e/ou **USART+TIM2** se acordado
- [ ] Pins SPI2 / I2C1 no draft sch se no escopo (`NOT FABRICABLE`)
- [ ] Goldens event-graph / prove com `diff` (RP, STM32 USART, SPI `expected_spi/`, e/ou I2C `expected_i2c/`)
- [ ] Design com preferência de fabricante
- [ ] Reconstruct: report `stop_reason` / estagnação (≠ auto-fix)
- [ ] Demo: `run.sh` / `pilot_stm32` / `run_w1_spi.sh` / `run_x3_i2c.sh` / `run_y3_triple.sh` / `run_z2_tim.sh` / `base hil` EXPERIMENTAL
- [ ] Relatório de viabilidade + handoff

## Não-escopo

- [ ] PCB fabricável / Gerber de fábrica
- [ ] Flash HIL “production” / silício sem aprovação escrita
- [ ] `libusb` obrigatório no CI do Cliente
- [ ] SaaS turnkey / 24×7
- [ ] Substituir gate RP pelo STM32
- [ ] Auto-fix completa via `reconstruct --continuous`
- [ ] Amiga/CD32 como entregável de release
- [ ] Claim “v1.0 = produto industrial completo” / ASIC drop-in

## HIL (add-on)

| Item | Default |
|------|---------|
| CLI | `base hil enumerate\|flash` — EXPERIMENTAL |
| Dry-run | `--mock-flash` → `mock_dry_run` |
| Real | feature `hil_programmer` + `ALLOW_FLASH` + CMD do operador |

## Aceite

1. Gates RP verdes (`run.sh` + `run_t1_b2.sh`) com goldens `diff`.  
2. Se STM32: `pilot_stm32/run.sh`; se SPI: `run_w1_spi.sh` (+ goldens `expected_spi/`); se I2C: `run_x3_i2c.sh`; se triple: `run_y3_triple.sh`; se TIM: `run_z2_tim.sh` — ou bloqueio honesto.  
3. Relatório sem claims da seção Não-escopo.
